Group Usage ​

Feature Description ​

The bot supports use in groups, but some features require private chat for privacy considerations.

Available Group Features ​

Free Proxy

  • Send /socks in groups to get free trial proxy
  • Suitable for temporary use or testing

Daily Check-in

  • Send /checkin or "checkin" in groups
  • Check-in rewards are the same as private chat
  • Coupons will be sent via private message (for privacy protection)

Invite Center

  • Send /invite or "invite" in groups
  • Will guide you to use full features in private chat

Rankings

  • Can view check-in and invite leaderboards in groups
  • Send /ranking_checkin or /ranking_invite

Private Chat Only Features ​

The following features are only available in private chat for privacy and security reasons:

  • Plan purchase and subscription links
  • Complete Telegram proxy node list
  • Detailed traffic query
  • Complete invite statistics

Add Bot to Group ​

  1. Add Bot to Group

    • Add bot in group settings
    • Or use /addtgroup command to get add link

  2. Set Administrator Permissions (Recommended)

    • Set bot as administrator
    • Keep default permissions
    • Can enjoy full features after setup

  3. Group Promotion Rewards

    • When group members register through your invite, you get rewards
    • When group members purchase plans, you get commission
    • Fully automatic promotion, no manual intervention needed

Notes ​

  • Some features in groups will guide you to use in private chat
  • It is recommended to set the bot as administrator for the best experience
  • Group promotion rewards are the same as personal invite rewards
