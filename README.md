# 快速开始

## 首次使用

1. **启动机器人**
   - 在 Telegram 打开[@OneTapLink_bot](http://t.me/OneTapLink_bot)
   - 点击「开始」或发送 `/start` 命令
   - 系统将自动为您创建账号

2. **绑定账号**
   - 首次使用时会自动注册账号
   - 账号信息会通过 Telegram ID 自动关联
   - 无需手动输入邮箱或密码

3. **开始使用**
   - 发送 `/start` 查看功能菜单
   - 使用底部快捷键盘快速访问功能
   - 或使用命令直接调用相应功能

## 功能概览

本机器人提供以下核心功能：

- 🚀 **套餐购买** - 购买 VPN/机场服务套餐
- 🛫 **Telegram 代理** - 一键设置 Telegram SOCKS 代理
- ✅ **每日签到** - 每日签到获得现金或优惠券奖励
- 💰 **邀请中心** - 邀请好友获得丰厚奖励
- 📊 **流量查询** - 实时查看流量使用情况
- 🏆 **排行榜** - 查看签到和邀请排行榜
